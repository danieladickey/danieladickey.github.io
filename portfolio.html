Tetris

Daniel wrote a Tetris-like game with JavaScript, Canvas, HTML, CSS, and Bootstrap. Some of the features of the game include sound, customizable controls, particle effects, high-score (stored in the browser), attract-mode. Attract mode is a simple algorithm that plays the game whengit no input is detected to attract a user to the game and demonstrate how it is played.

Maze

This was Daniel's first graphical game. The user can select the grid size before beginning the game. It uses Prim's algorithm to produce a random maze. It also utilizes a depth-first recursive algorithm to solve the maze. The solution is calculated each time the user moves so that the user can be provided with a hint of what move to make next or see the entire path from their current position to the end. The game was made using Javascript, Canvas, HTML, and CSS.
