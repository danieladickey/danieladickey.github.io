Maze

This was Daniel's first graphical game. The user can select the grid size before beginning the game. It uses Prim's algorithm to produce a random maze. It also utilizes a depth-first recursive algorithm to solve the maze. The solution is calculated each time the user moves so that the user can be provided with a hint of what move to make next or see the entire path from their current position to the end.

Bee AI

Using audio clips of ambient sound, crickets chirping or bees buzzing from three different data sets. Daniel trained one single convolutional neural network (CNN) on all three data sets. He trained three individual CNNs on one data set each and then used the three CNNs to create an ensemble. He tried five different types of ensembles. TFLearn was used to design the CNNs, pickle to save the models, and TensorFlow was used to train, validate, and test. The Random Forest used the sklearn library. The training was accomplished on Google Colab to utilize their graphics cards to speed up training. The CNN, five ensembles, and a random forest (RF) were all compared against each other to see who was the most accurate. After training the AIs, they were tested to see which could more accurately identify that type of audio clip correctly. Surprisingly the single CNN beat out all the ensembles and the RF.

Python, NumPy, Tensor Flow, TF Learn, pickle, Google Colab
Bus Tracking Android App

A native Android app for tracking the Cache Valley Transit District (CVTD) buses in Cache Valley, Utah. At regular intervals, XML data is retrieved and parsed from where CVTD stores the most recent GPS data online. The app updates the icons of the buses' positions on the map using Mapbox's SDK. Android OS provides the GPS location of the user to be displayed on the map. Banner ads are incorporated via Google Mobile Ads for app monetization. Now citizens of Cache Valley won't miss the bus ever again.
