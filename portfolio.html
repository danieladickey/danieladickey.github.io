Tetris

Daniel wrote a Tetris-like game in JavaScript, Canvas, HTML, & CSS. Some of the features of the game include sound, customizable controls, particle effects, high-score, attract-mode. Attract mode is a simple algorithm that plays the game when no input is detected to attract a user to the game and demonstrate how it is played.
